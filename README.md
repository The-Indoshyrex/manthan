# Manthan

Manthan is a Work Hub and Team Collaboration Platform.

## Features

- User Authentication
- User Profiles
- Feed System
- Teams
- Goals Management
- Tasks Management
- Notifications
- Messaging System
- Admin Dashboard

## Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

### Realtime
- Socket.io
